Hello from my CI/CD Demo Project 🚀

This project demonstrates a complete CI/CD pipeline hosted on AWS. It demonstrates how code flows from a GitHub commit all the way to a running application on EC2.

🔧 Services Used

  • ✅ GitHub for source control
  • ✅ CodePipeline to orchestrate CI/CD
  • ✅ CodeBuild for automated builds and tests
  • ✅ S3 bucket for storing build artifacts
  • ✅ CodeDeploy for deployment
  • ✅ CloudFormation for infrastructure as code
  • ✅ EC2 instance hosting this Flask app
  • ✅ IAM for secure access management

It all works together to continuously deliver code, beautifully and simply. ✨
